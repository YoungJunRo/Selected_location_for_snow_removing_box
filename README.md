# Selected_location_for_snow_removing_box
머신러닝을 활용한 은평구 제설함 추가 배치 위치 설정 프로젝트


## Model

### LSCP
- 동국대학교 산업시스템공학과 심미경 개발

### MCLP
gurobipy 와 mip를 활용한 최적화 모델
